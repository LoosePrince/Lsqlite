# 管理后台使用说明

## 登录

使用环境变量中的 `ADMIN_USER` 和 `ADMIN_PASSWORD` 登录后台。

开发环境默认地址：`http://localhost:5173`。

生产构建后默认由 Node.js 服务提供后台页面。

## 创建数据库

创建数据库时需要填写：

- 数据库名称。
- 可选 key。如果留空，系统生成随机 key。
- 可选备注。

创建完成后会显示数据库 key。服务端只保存 key 摘要，不保存明文 key，请立即保存。

## 管理数据库

后台支持：

- 查看数据库文件路径、状态、创建时间和最后访问时间。
- 查看表、视图、字段和索引。
- 执行 SQL。
- 禁用或启用数据库。
- 轮换 key。

## SQL 控制台

管理员 SQL 控制台用于维护数据库。管理员可以执行更高权限语句，因此执行破坏性 SQL 前应先确认目标数据库和影响范围。

## key 轮换

轮换 key 后旧 key 立即失效。调用方需要更新 `Authorization: Bearer <database_key>`。
